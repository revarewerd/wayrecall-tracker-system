  **Задача 1: Two Sum (Индекс двух сумм)**
  ```
  Дано: массив целых чисел nums и целое число target
  Найти: индексы двух разных элементов, которые дают target
  Условие: O(1) space или O(n) time, не можно одно число использовать дважды
  Пример: nums = [2,7,11,15], target = 9 → [0,1]
  ```

  **Задача 2: Longest Substring Without Repeating Characters**
  ```
  Дано: строка s
  Найти: длину самой длинной подстроки БЕЗ повторяющихся символов
  Пример: s = "abcabcbb" → 3 (подстрока "abc")
  Пример: s = "bbbbb" → 1 (подстрока "b")
  ```

  **Задача 3: Median of Two Sorted Arrays**
  ```
  Дано: два отсортированных массива nums1 и nums2
  Найти: медиану их объединённого массива
  Сложность: O(log(min(m,n)))
  Пример: nums1 = [1,3], nums2 = [2] → 2.0
  ```

  **Задача 4: Trapping Rain Water**
  ```
  Дано: массив высот elevation map [0,1,0,2,1,0,1,3,2,1,2,1]
  Найти: сколько воды может собраться ПОСЛЕ дождя
  Условие: O(n) time и O(1) space (если возможно)
  Пример: вода собирается между высотами
  ```

  **Задача 5: Container With Most Water**
  ```
  Дано: массив height, где height[i] = вертикальная линия в позиции i
  Найти: максимальную площадь КОНТЕЙНЕРА между двумя линиями
  Площадь = min(height[i], height[j]) * (j - i)
  Условие: O(n) time
  ```

  - Результат: Список готов для дороги
